# my-css-library

A CSS Library I'm working on to learn LESS and PostCSS. Borrowing a lot from Tachyons

Tachyons github: https://github.com/tachyons-css/tachyons/

---

## Install

```bash
yarn install
```

Or

```bash
npm install
```

---

## Compile

```bash
yarn run build
```

or

```bash
npm run build
```
